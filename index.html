I'm Dejón ( pronounced \ˈdā - jän \ ), but everyone calls me DJ. The J stands for Jzábar, if you were wondering.

I'm a full stack software engineer specializing in web development. That means different things to different people. To me it means that I choose the right technologies for projects, code up solutions to problems, I am always learning and I'm not afraid of CSS! Indeed, this site is styled with basic CSS, no pre-processors or external libraries. I used jQuery when creating the mobile navigation animation. I love vanilla Javascript, but sometimes you just don't feel like typing document.getElementById... :)

My favorite languages to work with are JavaScript and Go. I'm a fan of TypeScript because strong typing is useful, and valid JavaScript is valid TypeScript, which means teams can ease into using TypeScript. I like Sass for similar reasons, valid CSS can be valid .scss. I prefer MySQL to MongoDB, React is my front-end frame work of choice, and if you ever want to talk Redux and RxJS send me a tweet!

Growing up, I remember it being a big deal if someone's parent's car had a car phone, and I had a collection of AOL start up discs that filled a CD tower. Having watched the internet grow from Napster and Geocities to what it is today, ( I had a Palm Pre! ), I have remained constantly fascinated with the Web, how it works, and how it influences our lives.

My spare time is generally spent doing something outside, or contibuting to the communities that I belong to. I mentor at NodeSchool San Francisco and NodeSchool Oakland.

I've been to 18 of the 50 US States, 17 of the 59 National Parks, and too many State, County and City parks to count. I love to go cycling, camping, backpacking, and snowboarding.

I have two dogs, Chloe the Australian Cattle Dog, and Cyndi the Chihuahua/Pug. I'm running out of things to say, so I'm going to include some lorem ipsum. Just to fill out the page, so I can use this cool parallax effect.
